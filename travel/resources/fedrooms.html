Hotels (FedRooms)

FedRooms provides Federal Travel Regulation (FTR)-compliant hotel rooms for federal government travelers while on official business. More FedRooms information (link to full site)

Online reservations can be made using https://www.fedrooms.com/ (a non-government website, not optimized for mobile)

Travelers MUST ASK for the FedRooms rate by name: Rate code XVU
